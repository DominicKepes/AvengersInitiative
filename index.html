Description

Avengers Initiative is a simple webpage for Marvel characters or further information. You can type a characters name in the search box and press marvel to recieve marvels official description from their api. Also, you can type anything and click wikipedia for a basic wiki search. Typing a search in the top right box and clicking Wikipedia or Marvel buttons will pull information from the respective API. It will propagate the page with the heroes name, image, and description. You have to be careful to type it with correct spacing/punctuation as the apis are very particular. The wikipedia API also has some things named inccorectly, so you may get wrong results. Not all wikipedia searches will pull an image.
